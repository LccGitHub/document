# 给SD卡分区过程

## 要求
现有一张16G的SD卡，需要划分三个分区，分区如下

1. var ：60M
2. cache:500M
3. data：剩余

## 操作如下：

#### 第一：制作分区

### 第二：将分区刷成ext4格式
重新插拔取卡器，然后执行以下命令
    
    sudo mkfs.ext4 -L var /dev/sdc1
    sudo mkfs.ext4 -L cache /dev/sdc2
    sudo mkfs.ext4 -L data /dev/sdc3
